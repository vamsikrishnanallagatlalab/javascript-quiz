# javascript-quiz


A modern, fully-featured quiz application built with **vanilla HTML, CSS, and JavaScript** — no frameworks required. Designed for developers, learners, and educators who want a sleek, responsive, and intuitive quiz experience.

 **Live Demo**: https://vamsikrishnanallagatlalab.github.io/javascript-quiz/


##  Features

-  **Dark & Light Mode** — toggle with sun/moon icon, respects system preference  
-  **Question Sidebar** — desktop sidebar + mobile offcanvas navigation  
-  **Progress Tracking** — visual progress bar + question counter  
-  **Instant Feedback** — color-coded answers ( correct /  incorrect)  
-  **Smart Scoring** — 5-tier feedback based on performance:
  - `0–4` → *Bad luck — try again!*  
  - `5–9` → *OK score — give it another try*  
  - `10–15` → *Really Good Score but could do better*  
  - `16–19` → *Great Job!*  
  - `20` → * Congrats! You got full points!*  
-  **Fully Responsive** — works on mobile, tablet, and desktop  
-  **No external dependencies** — pure Bootstrap 5 + vanilla JS  
-  **Fast & Lightweight** — < 50 KB total (HTML + CSS + JS)
